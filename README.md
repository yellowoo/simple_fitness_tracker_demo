# Align · 左侧力线康复打卡 App

基于你的《左侧骨盆三维扭转康复训练完整计划》制作的 PWA。

## v1.2 新增 · 分组打卡

### 训练动作不再是一次打勾

之前"蚌式 左 15 + 右 8 × 3 组"必须做完 3 组才能打卡,现在拆成 3 次操作:

- **今日页训练动作右侧显示进度按钮**:`0/3` → `1/3` → `2/3` → 完成
- **每点一次 +1 组**,有触觉反馈(支持震动的设备)
- **完成最后一组时自动弹出质量分**(1-3 分)
- **点击质量分立即完成**,不再需要"完成"按钮 —— 视觉确认 250ms 后自动关闭
- **半途想取消**:质量分弹窗下滑或点背景即可关闭(此时所有组已记录,只是没填质量分)
- **想重置当天进度**:点击已完成的进度按钮会询问是否重置;长按未完成的进度按钮也可重置
- **训练动作的清单上**显示"15 次/组"等每组目标提示

### 编辑模式可调每组次数和组数

详情页 ✎ 编辑模式下,训练 / 拉伸类动作多了"分组打卡设置"区:

- **每组目标**:文本输入,如 "12 次"、"30 秒"、"左 15 + 右 8"
- **组数**:数字输入框,1-20 组之间
- **修改后立即生效**:今日页进度按钮的总数会同步更新;如果你已经打了 3 组,改成 2 组,会自动 clamp 到 2 组完成

### 哪些动作适用?

任何 duration 里包含"× N 组"或"N 组"的动作。例如:
- 蚌式(左 15 + 右 8 × 3 组)→ 进度按钮 0/3
- 死虫(12 次 × 3 组)→ 进度按钮 0/3
- 半跪姿髋屈肌拉伸(90 秒 × 2 组)→ 进度按钮 0/2

单组动作(松解类、晨间)还是普通的圆形 checkbox,行为不变。

### 数据兼容性

之前 v1.1 打过的卡(只有 done:true)向后兼容 —— 自动视为"已完成所有组",不会丢历史。

## v1.1 功能 · 编辑 + 计时器(回顾)

### 1. 动作详情编辑功能

详情页右上角 ✎ 铅笔按钮进入编辑模式,可改名称、时长、工具、体位、要领、感受、警示。
节点(松解路径)可增删改、上下排序。
修改即保存,同步到所有日期。
我的页有"恢复所有动作默认值"批量重置。

### 2. 计时器与闹钟

松解 / 拉伸 / 晨间类动作详情页底部有计时器:

- **大字时间显示**,屏幕常亮(Wake Lock API)
- **节点提醒预设**:无 / 每 30 秒 / 每 60 秒 / 对半切换 / 按节点
- **完成全屏闹钟** + 持续蜂鸣(每 1.5 秒响 3 声)

### 计时器使用建议

| 动作 | 推荐预设 | 推荐时长 |
|---|---|---|
| 左内收肌(3 段路径) | 按节点 | 4 分钟 |
| 左髂腰肌 | 每 60 秒 | 2 分 30 秒 |
| 左 TFL + 股直肌 | 按节点 | 3 分钟 |
| 左腰方肌 + 背阔肌 | 按节点 | 5 分钟 |
| 左臀中肌 + 梨状肌 | 按节点 | 3 分钟 |
| 左腘绳肌下段 | 每 30 秒 | 2 分钟 |
| 半跪姿髋屈肌拉伸 | 对半切换 | 3 分钟 |

## ⚠️ iOS 闹钟使用注意

由于 iOS PWA 的限制:

1. **必须保持前台**:闹钟是 Web Audio 合成,锁屏会停止响铃 → 默认开启屏幕常亮
2. **首次需手动开启声音**:启动计时器点"开始"= 用户交互 = 才能播声音
3. **静音键不影响**:Web Audio 不受静音键影响,放心睡前用
4. **添加到主屏幕后效果最佳**

1. 新建 public 仓库,上传所有文件到根目录
2. Settings → Pages → Source 选 main 分支
3. iPhone Safari 访问,分享 → 添加到主屏幕

## 数据存储

所有数据存在 iPhone 的 Safari localStorage 里,分两块:
- `align_data_v1` — 打卡记录、感受日记、当前阶段
- `align_edits_v1` — 动作编辑(含计时器配置、每组次数、组数)

导出备份会同时导出这两块。

## 核心交互流程

### 完成一次松解(以内收肌为例)
1. 今日页点"左侧内收肌松解" → 详情
2. 滑到底部计时器,点"开始"(屏幕常亮 + 激活音频)
3. 倒计时显示 + 节点提示
4. 时间到,全屏红色闹钟 + 持续响铃
5. 点"关闭" → "标记完成"

### 完成一次训练(以蚌式为例)
1. 今日页"蚌式"右侧看到 `0/3` 按钮
2. 做完第一组 → 点 `0/3` → 变 `1/3`
3. 做完第二组 → 点 → `2/3`
4. 做完第三组 → 点 → 自动弹质量分
5. 点 1/2/3 → 250ms 后自动完成

### 编辑动作
1. 任意动作 → 右上角 ✎
2. 字段变可编辑,改完直接退出(自动保存)
3. 训练 / 拉伸类可改每组次数和组数
